# LEDDMX BLE

Home Assistant integration for LEDDMX BLE LED strips. Supports automatic discovery, RGB color, brightness, and effects.

## Features

- Automatic BLE discovery (no connection during setup)
- RGB color control
- Brightness
- Effects
- Supports all LEDDMX devices (LEDDMX-03-*, etc.)

## Requirements

- Home Assistant with Bluetooth support
- Bluetooth integration configured
